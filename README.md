# React Dashboard App

## Setup

```bash
yarn install
yarn start
```

## Running tests

```bash
yarn test
```

## Project Structure
- `src/components/` - Components like Sidebar
- `src/hooks/` - Custom React Hooks
- `src/pages/` - Pages like Dashboard, Blogs, Post Detail
- `src/services/` - API setup
- `src/styles/` - Global styles
- `src/tests/` - Unit tests
